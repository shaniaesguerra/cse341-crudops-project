# CSE 341 Project 2 CRUD Operations

Project performs CRUD operations on a MongoDB database, be published to the web through Render, 
incorporate security measures, and include API documentation with the use of Swagger. This is made using node.js

App uses the following dependencies:
- body-parser
- dotenv
- express
- mongodb
- mongoose
- swagger-autogen
- swagger-ui-express

Link to the project:
